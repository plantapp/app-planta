About App

PlantAPP


Es una aplicación para la recopilación, anotación y recuperación de imágenes para ayudar en la identificación de plantas. Se incluye un sistema de apoyo para la identificación automática de las plantas a partir de imágenes en comparación con las imágenes de una base de datos botánica. Los resultados se utilizan para obtener el nombre botánico de una planta, si esta es suficientemente ilustrada en la base.
